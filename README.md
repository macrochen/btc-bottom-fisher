# BTC Bottom Fisher (比特币“黄金坑”监控仪)

🚀 **完全自动化的比特币八维全景底部监控、逃顶预警与山寨季大屏** 

本项目是一个部署在 [Cloudflare Workers](https://workers.cloudflare.com/) 上的无服务器量化追踪程序。它通过整合**多个免费的公开 API**，自动计算比特币的极致超跌信号，并为您提供“闭眼买入”级别的底层抄底提示。

## 🌟 核心特色

- **零成本运行**：利用 Cloudflare Workers 的免费额度，无需购买服务器。
- **无需付费 API**：不依赖昂贵的链上数据订阅（如 Glassnode 或 CryptoQuant），而是利用数学推导和免费 API (Binance, CoinGecko, Alternative.me, BGeometrics) 还原高精度的底层数据。
- **八维全景雷达模型**：同时监控均线偏离、情绪冰点、矿工底线、估值中枢（MVRV/NUPL/SOPR），甚至包含宏观牛熊指数（CBBI）和逃顶预警（Pi Cycle Top）。
- **山寨季预警**：内建基于市值前 20 主流山寨币表现的“山寨季指数”计算引擎，为您解决“买大饼还是买山寨”的资产配置难题。
- **赛博朋克大屏**：自带开箱即用的暗黑科技风 Dashboard，信号触发一目了然。

## 📊 监控指标与阈值详解

1. **60日均线极度偏离**
   - **逻辑**：均线是用来算账的。当当前价格大幅度跌破60天前的平均持仓成本时，意味着短线情绪彻底崩溃。
   - **阈值**：偏离度 `< -15%` 触发。
2. **恐慌与贪婪指数 (Fear & Greed Index)**
   - **逻辑**：忽略贪婪，只盯恐慌。市场进入绝望期时，主力正在吸筹。
   - **阈值**：指数 `< 30` 触发。
3. **日线超卖确认 (RSI 14)**
   - **逻辑**：极佳的波段过滤器，历史证明 90% 的日线级别严重超卖都会迎来强力反弹。
   - **阈值**：日线 RSI `< 35` 触发。
4. **普尔倍数矿工底线 (Puell Multiple)**
   - **逻辑**：衡量每日比特币产出价值与过去365天平均值的比率。低于 0.5 意味着全网矿工在痛苦割肉。
   - **计算技巧**：利用减半周期内每日固定产量（~450 BTC）的特性，在程序中将其等价简化为 `现价 / 365日均线`，实现免 API key 的高精度模拟。
   - **阈值**：普尔倍数 `< 0.5` 触发。
5. **MVRV (市值 / 已实现市值)**
   - **逻辑**：衡量全网比特币当前总市值与所有比特币最后一次移动时的价格总和的比率。跌破 1 代表整体浮亏，是长周期绝佳的建仓区。
   - **数据源**：接入 BGeometrics API。
   - **阈值**：MVRV `< 1.0` 触发。
6. **NUPL (净未实现利润/亏损)**
   - **逻辑**：衡量全网筹码的净未实现利润与亏损状态。跌破 0 意味着大面积亏损，恐慌情绪达到冰点。
   - **数据源**：接入 BGeometrics API。
   - **阈值**：NUPL `< 0` 触发。
7. **SOPR (已花费输出利润率)**
   - **逻辑**：计算当日在链上转移的 BTC 相比上次移动的盈亏情况。持续小于 1 说明人们在割肉。
   - **数据源**：接入 BGeometrics API。
   - **阈值**：SOPR `< 1.0` 触发。
8. **CBBI (综合牛市指数)**
   - **逻辑**：整合多个顶级链上指标的综合得分 (0-100)。
   - **数据源**：ColinTalksCrypto API。
   - **阈值**：底部 `< 15` 触发，顶部 `> 80` 逃顶告警。

## 🚨 逃顶预警器 (Top Warnings)

系统内置了逃顶专属预警，当进入狂暴牛市末期时会触发红色警报：
1. **Pi Cycle Top 预警**
   - **逻辑**：当短期均线 (111日) 疯狂拉升，上穿长期均线乘数 (350日×2) 时，历史周期均在此附近见顶。纯净的价格行为预警器。
2. **CBBI 牛熊指数超买**
   - **逻辑**：当得分突破 80 分时，说明市场极度狂热，随时可能崩盘。

## 💡 终极决策：山寨币季节指数

当底层指标触发“黄金坑”时：
- **山寨指数处于高位 (>75)**：山寨币群魔乱舞，泡沫严重，此时抄底应**优先买入比特币避险**。
- **山寨指数处于低位 (<25)**：山寨币泡沫挤干，此时抄底应**重点布局市值前 20 的主流山寨币**，获取更高弹性。

## 🛠️ 安装与部署

本项目使用 TypeScript 和 [Hono](https://hono.dev/) 框架开发。

### 1. 本地环境准备
克隆代码并安装依赖：
```bash
git clone https://github.com/macrochen/btc-bottom-fisher.git
cd btc-bottom-fisher
npm install
```

### 2. 本地开发与测试
```bash
npm run dev
```
运行后，Wrangler 会启动一个本地开发服务器（默认 `http://localhost:8787`），直接在浏览器打开即可查看数据大屏。

### 3. 配置 MVRV 专属 API Key (BGeometrics)
MVRV 数据依赖 [BGeometrics](https://bgeometrics.com/) 提供的 API。请注册并获取您的免费 API Key，然后通过 Wrangler 将其配置到 Secrets 环境变量中：
```bash
npx wrangler secret put BGEOMETRICS_API_KEY
# 终端提示输入时，粘贴您的 API Key
```
如果您在本地开发，可以创建一个 `.dev.vars` 文件并写入：
```
BGEOMETRICS_API_KEY=您的API_KEY
```

### 4. 一键部署到 Cloudflare
```bash
npm run deploy
```
Wrangler 将会引导您登录 Cloudflare 账号，并自动帮您部署。部署成功后，您将获得一个属于自己的 `*.workers.dev` 免费域名，您可以将其收藏到手机桌面，每天查看。

## 📺 策略来源与原理解读

本项目开发的核心思想来源于如下视频的总结：
🔗 [YouTube 参考视频](https://www.youtube.com/watch?v=weKp2-4i3AQ)

> 在喧嚣的币圈，最让人焦虑的不是赚不到钱，而是总觉得自己盯盘不够紧、操作不够多。但实际上，真正能让人稳健翻倍的策略，恰恰是“反勤奋”的。现货交易的核心逻辑不是去捕捉每一次市场波动，而是像一个极耐心的猎人，只在市场把价格送到绝对便宜的“黄金坑”时，才轻轻扣动扳机。
>
> ### 放弃捕捉每天的行情，只赚“好赚的钱”
> 大多数人亏钱，是因为既想要高胜率，又想要高频交易，还想要惊人的盈亏比。这在客观规律上是不可能实现的。
> 中国A股的传奇人物徐翔，拆解其惊人的长期收益后会发现，他每年的平均收益其实并不足翻倍。投资的秘密不在于爆发力，而在于长期积累并极力减少亏损。只赚简单、好赚的钱，把持仓周期定在两周到几个月不等，在一轮牛熊周期里，就能跑出非常可观的滚雪球效应。
>
> ### 寻找“黄金坑”的八维共振坐标
> 要做到不盯盘也能精准抄底，需要利用几个经典的量化指标来寻找市场的极端错误定价。当以下几个维度中有两到三个同时拉满时，就是闭眼买入的信号。
>
> ### 均线扣减价的极端偏离
> 均线不是用来追涨的，而是用来算账的。通过观察60日均线的抵扣价（即60天前的收盘价），当当下的价格大幅度低于60日抵扣价，导致其指标波动呈现出极低的负值（例如负17甚至负28）时，意味着市场在短期内已经超跌、情绪彻底宣泄。这是技术面上的绝对低位。
>
> ### 恐慌指数的冰点买入法
> 在贪婪与恐慌指数中，我们应该彻底忽略贪婪，只死盯着恐慌。当市场跌到人人自危、指数低于30时，必须开始关注；低于20时，要密切注视；而一旦指数跌破10，市场进入极度恐慌的绝望期，主力的筹码往往在这个时候默默换手，这时候直接买入，胜率极高。
>
> ### 普尔倍数的矿工底线
> 普尔倍数（Puell Multiple）衡量的是每天比特币产出价值与过去365天平均值的比率。这是一个未来几十年都不会失效的底层逻辑，因为矿工的生产成本是有硬性法币开支的。当这个指标低于0.5、进入绿色区间时，意味着全网矿工都在痛苦割肉，而历史证明，每一次这个指标触绿，都是比特币极其罕见的周期大底部。
>
> ### 日线RSI的超卖确认
> 作为日常辅助，日线级别的 RSI 14 是个极佳的过滤器。只要日线RSI跌破35，甚至触及30以下，市场就已经进入严重的超卖状态。历史上90%的日线超卖，随后都会迎来一波至少20%以上的强力反弹。
>
> ### 终极决策：买大饼还是买山寨？
> 当上述指标产生共振、确定了绝佳的买点之后，该把资金投向哪里，取决于“山寨币季节指数”（Altcoin Season Index）。
> 这个指数如果运行在高位，意味着山寨币已经群魔乱舞，此时买点出现应该果断买入比特币避险；反之，如果该指数处于极低的底部，说明山寨币的泡沫已经被挤得干干净净，此时配合买点出现，重点布局市值前20的主流山寨币，能获得远超比特币的翻倍弹性。
>
> ### 再想一层
> 这套战法之所以有效，建立在一个至关重要的前置条件上：比特币在超大周期（如月线级别）上，依然处于由其底层通缩算法决定的“永恒牛市”和法币贬值的趋势中。
> 读者最容易产生的误解是，把这套针对现货、旨在捕捉周期性“黄金坑”的指标，错误地应用到带有高杠杆的合约交易中。在极端行情中，即便指标提示了绝对底部，合约也极有可能在价格正式反转前的最后一次“插针”中爆仓归零。
> 这套方法的本质是用时间换空间。它的成立边界在于：你必须能够忍受一年中可能只有廖廖几次的出手赞机，以及买入后可能需要装死数周的寂寞。
> 建议在实际操作中，将 LookIntoBitcoin 等数据工具加入浏览器收藏夹，设置好雷达监控，在信号未出现前保持绝对的空仓耐性。
